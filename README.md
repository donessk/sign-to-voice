# Sign to Voice — S2V1

> *Inclusive Innovation — a world where everyone can be heard*

**Sign to Voice** is a Progressive Web App that translates Ghanaian Sign Language gestures into spoken words in real time.

Built by **Gesture Squad**.

---

## First Time Setup

This page is used to initialise the app before using it with the Sign to Voice glasses unit.

**Steps:**
1. Open this page on any device with internet
2. Tap **Connect** — MediaPipe (~3MB) downloads and caches on your device
3. Done — the app now works fully offline

**Daily use:**
1. Connect to the **SignToVoice** WiFi hotspot
2. Open `http://sign.voice` or `http://192.168.4.1`
3. Tap **Connect** — gesture translation starts instantly
